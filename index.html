プロジェクト課題について

このHTMLは、「世界の天気」の検索結果を1件表示するためのHTMLです。見出しで検索結果が1件であることを示し、その下にリストを使って天気に関する詳細情報を整理しています。内容には、緯度・経度といった位置情報、天気の状態、気温（現在・最高・最低）、湿度、風向や風速などの気象データが含まれています。また、天気アイコンの画像や都市（東京）の地図画像も表示されており、視覚的にも情報を補足しています。これにより、利用者は東京の現在の天気状況を直感的に把握できる構成になっています。

プロジェクト課題で作成したページ：project.html

各授業回の学習事項

第1回 WWW と HTML

Webページを提供する仕組みのことをWWWという。WWWを構成する技術要素として、「プロトコル」、「サーバ」、「パス」がある。HTMLはWebページの構造を記述するための書式。HTML文書を作ることによって、我々はWebページに掲載するコンテンツと、ページの基本的なレイアウトを設定できる。

演習の成果：element.html

第2回 CSS による Webページの装飾

CSSを用いることにより、ページをより見やすく装飾することができる。CSSをHTMLに適用する方法は、「外部スタイルシート」、「内部スタイルシート」、「インラインスタイル」の3つがあるが、「インラインスタイル」は装飾したい要素全てに同じstyle属性をコピペする必要があり、それらの設定を微調整したい場合、コピペした記述をすべて編集する必要が生じるため避けるべきである。

演習の成果：browser-intro.html

第3回 JavaScript の基本

JavaScriptはWebページ上で複雑な機能を実現するプログラミング言語で、Webページを開いた直後や、ボタンを押したときにJavaScriptプログラムが実行される。ページの自動更新やアニメーションなど動きのあるページを実現するためにJavaScriptプログラムを使う。Webページの開発者ツールを用いてデバックやJavaScriptプログラムの動作確認などができる。
